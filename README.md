# signalk-update

Update/upgrade plugin for [SignalK Node Server](https://signalk.org/).

**Status:** skeleton — name reserved on npm, implementation pending.

## Planned scope

Surface and manage SignalK server updates from inside the server:

- Detect available updates for the SignalK server itself
- Detect available updates for installed plugins
- One-click apply with safe rollback
- Update channel / pinning controls
- Pre-update health checks (works well alongside `signalk-doctor`)

## Install

Not ready for use. Once published with real functionality, install via the SignalK Appstore as `signalk-update`.
